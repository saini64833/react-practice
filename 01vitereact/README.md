# 01 Vite React

A basic React project setup using Vite as the build tool.

## Description

This is a foundational Vite + React project created for learning React fundamentals and understanding how Vite improves the development workflow with faster build times and hot module replacement (HMR).

## Features

- ⚡ Fast development experience with Vite
- ⚛️ React for building user interfaces
- 🔄 Hot Module Replacement (HMR) support

## Project Structure

```
01vitereact/
├── public/          # Static assets
├── src/             # Source code
│   ├── App.jsx      # Main App component
│   ├── main.jsx     # Entry point
│   └── assets/      # Image and media assets
└── vite.config.js   # Vite configuration
```

## Installation

1. Navigate to the project directory
2. Install dependencies:
   ```bash
   npm install
   ```

## Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port displayed in your terminal).

## Build

To create an optimized production build:

```bash
npm run build
```

## Preview

To preview the production build locally:

```bash
npm run preview
```

## Technologies Used

- **Vite** - Build tool and dev server
- **React** - Frontend library
- **JavaScript** - Programming language

## Learning Goals

This project helps in understanding:

- Vite project structure and configuration
- React component basics
- Development workflow with Vite
