# 02 Counter

A simple counter application built with React to demonstrate state management and event handling.

## Description

This React application showcases the fundamentals of managing component state with `useState` hook and handling user interactions through button clicks. It's a great learning project for understanding React's reactive nature.

## Features

- ✅ Increment and decrement counter functionality
- 📊 Real-time counter display
- 🎯 Simple and intuitive UI

## Project Structure

```
02counter/
├── public/          # Static assets
├── src/
│   ├── App.jsx      # Main counter component
│   ├── main.jsx     # Entry point
│   ├── assets/      # Image and media assets
│   └── index.css    # Styling
└── vite.config.js   # Vite configuration
```

## Installation

1. Navigate to the project directory
2. Install dependencies:
   ```bash
   npm install
   ```

## Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port displayed in your terminal).

## Build

To create an optimized production build:

```bash
npm run build
```

## Preview

To preview the production build locally:

```bash
npm run preview
```

## Technologies Used

- **React** - Frontend library
- **Vite** - Build tool
- **CSS** - Styling

## Learning Goals

This project demonstrates:

- React's `useState` hook for state management
- Event handling in React
- Component rendering and re-rendering
- Basic counter logic
