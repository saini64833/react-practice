# Tailwind Props

A React project demonstrating component props usage with Tailwind CSS styling for rapid UI development.

## Description

This project showcases best practices for creating reusable React components that accept props and using Tailwind CSS for styling. It's designed to illustrate how to build flexible, maintainable UI components with Tailwind's utility-first approach.

## Features

- 🎨 Tailwind CSS for styling
- 🔄 Reusable component patterns with props
- 📦 Component composition
- 🎯 Responsive design
- ⚡ Utility-first styling approach

## Project Structure

```
talwindProps/
├── public/                  # Static assets
├── src/
│   ├── component/          # Reusable components
│   ├── App.jsx             # Main application component
│   ├── main.jsx            # Entry point
│   ├── assets/             # Image and media assets
│   └── index.css           # Global styling
└── vite.config.js          # Vite configuration
```

## Installation

1. Navigate to the project directory
2. Install dependencies:
   ```bash
   npm install
   ```

## Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port displayed in your terminal).

## Build

To create an optimized production build:

```bash
npm run build
```

## Preview

To preview the production build locally:

```bash
npm run preview
```

## Technologies Used

- **React** - Frontend library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework

## Component Props

This project demonstrates:

- Passing props to components
- Destructuring props in functions
- Using props for conditional rendering
- Styling components with Tailwind classes passed as props
- Creating flexible, reusable components

## Learning Goals

This project is designed to teach:

- React component architecture
- Props usage and best practices
- Tailwind CSS utility classes
- Creating DRY (Don't Repeat Yourself) components
- Building scalable UI systems
- Component reusability patterns
- Responsive design with Tailwind CSS
